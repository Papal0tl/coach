# Rubric — Spiral Matrix

## Skill Targets

- Boundary simulation: correctly track top/bottom/left/right shrinking bounds.
- Direction cycling: traverse right → down → left → up and repeat.
- Termination: stop exactly when all m×n elements are collected.
- Edge cases: single row, single column, 1×1 matrix.
